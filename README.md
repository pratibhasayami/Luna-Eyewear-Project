# LUNA Eyewear App – UX Design Project (Work in Progress)

This project is part of my UX design learning journey based on the Google UX Design Certificate framework.

## Status
Currently in the Empathize phase.

## Project Goal
Design a mobile app for LUNA, an eyewear boutique in Nepal, to help users browse frames, manage prescriptions, and book eye appointments digitally.

## Completed so far
- User research approach (proto-personas)
- 3 user personas:
  - Jenny Pradhan (Fashion-conscious professional)
  - Samy Shrestha (Busy parent)
  - Abhi Shakya (Style seeker)

## Next steps
- Define phase (problem statements, insights)
- Ideation (HMW questions)
- Wireframes (low-fidelity designs)
- Lo-fi prototype in Figma
